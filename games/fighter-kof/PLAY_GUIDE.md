# 🥊 拳皇格斗游戏 - 操作指南
**项目**: fighter-kof
**版本**: 1.0
**游戏类型**: 双人格斗对战

## 🎮 游戏简介

这是一个基于 HTML5 Canvas 开发的双人格斗游戏，灵感来源于经典游戏《拳皇》（King of Fighters）。游戏支持本地双人对战，采用经典的三局两胜制制。

### 核心特色
- 🥊 **双人对战** - 两名玩家同一台设备对战
- ⚡ **能量系统** - 攻击积攒能量，释放强力必杀
- 💥 **连击系统** - 连续攻击触发炫酷连击
- 🛡️ **防御机制** - 反向移动减少伤害
- 🏆 **三回合制** - 三局两胜决出胜负

---

## 🕹️ 操作说明

### 玩家 1（Player 1）
左侧红色角色，使用 **WASD** 键位

| 按键 | 功能 | 说明 |
|------|------|------|
| W | 跳跃 | 向上跳跃，可进行空中攻击 |
| A | 左移 | 向左移动（向右移动时可防御） |
| D | 右移 | 向右移动（向左移动时可防御） |
| S | 蹲下 | 降低高度，躲避高段攻击（待实现） |
| J | 拳攻击 | 快速出拳，伤害中等 |
| K | 脚攻击 | 回旋踢，伤害较高（当前同拳攻击）|
| L | 必杀技 | 消耗50能量释放强力攻击 |

### 玩家 2（Player 2）
右侧蓝色角色，使用 **方向键** 键位

| 按键 | 功能 | 说明 |
|------|------|------|
| ↑ | 跳跃 | 向上跳跃，可进行空中攻击 |
| ← | 左移 | 向左移动（向右移动时可防御） |
| → | 右移 | 向右移动（向左移动时可防御） |
| ↓ | 蹲下 | 降低高度，躲避高段攻击（待实现） |
| 1 | 拳攻击 | 快速出拳，伤害中等 |
| 2 | 脚攻击 | 回旋踢，伤害较高（当前同拳攻击）|
| 3 | 必杀技 | 消耗50能量释放强力攻击 |

## ⚔️ 战斗技巧

### 基础连击
```
拳 → 拳 → 拳 (3连击)
拳 → 拳 → 必杀技 (强连击)
```

### 防御技巧
- **移动防御**: 向对手移动方向相反的方向移动可触发防御
- **伤害减免**: 防御时受到的伤害减少 80%
- **策略**: 观察对手动作，及时防御

### 能量管理
- 攻击命中:** +10 能量**
- 需要能量:** 50** 可释放必杀技
- 最大能量:** 100**
- 策略: 先积攒能量，再爆发必杀

### 连击策略
1. 靠近对手
2. 快速连续攻击
3. 触发连击后接必杀技
4. 大量消磨对手血量

## 🎯 游戏规则

### 胜利条件
- **KO**: 对手血量为 0
- **时间到**: 对手血量比己方少
- **平局**: 随机判定胜负

### 回合制
- 每局时间: 99 秒
- 回合赛制: 三局两胜
- 回合间休息: 自动重置

### 血量系统
- 初始血量: 100
- 普通攻击伤害: 15
- 防御伤害: 3 (15 × 20%)
- 必杀技伤害: 40

### 能量系统
- 初始能量: 0
- 最大能量: 100
- 攻击获得: +10
- 必杀消耗: 50

## 💡 高级玩法

### 技术连招
```
1. 跳跃攻击
   W → 跳跃
   在空中接近对手时
   J → 空中拳
   
2. 接地连击
   落地瞬间
   A → 贴近
   J → 拳 × 3
   L → 必杀技终结

3. 防守反击
   对手攻击时
   反方向移动 → 防御
   J → 立即反击
```

### 心理战
1. 假动作：移动后停止，引诱对手攻击
2. 能量威慑：保持能量满50，逼迫对手防御
3. 时机把握：等待对手攻击结束后的硬直时间
4. 预判跳跃：对手跳起时，提前预判落点

## 🚀 快速开始

### 1. 打开游戏
```
浏览器打开: /root/mygame/games/fighter-kof/index.html
```

### 2. 选择模式
- 点击"开始游戏"进入对战
- 点击"操作说明"查看控制

### 3. 开始对战
- P1 使用 WASD + JKL
- P2 使用 方向键 + 123

### 4. 观察UI
- 血条显示生命值
- 能量槽显示必杀能量
- 计时器显示剩余时间
- 连击数字显示连击数
