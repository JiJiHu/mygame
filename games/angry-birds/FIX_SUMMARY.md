## 🚀 如何运行游戏

### 方法 1: 直接打开（推荐）
1. 在浏览器中打开 `/root/mygame/games/angry-birds/angry-birds.html`
2. 需要网络连接（加载 Matter.js CDN）

### 方法 2: 本地服务器（开发用）
```bash
cd /root/mygame
python3 -m http.server 8000
```
然后在浏览器访问:
```
http://localhost:8000/games/angry-birds/angry-birds.html
```

### 方法 3: 离线使用（无需网络）
1. 下载 Matter.js 本地文件:
```bash
wget https://cdnjs.cloudflare.com/ajax/libs/matter-js/0.19.0/matter.min.js -O matter.min.js
```

2. 修改 HTML 链接:
```html
<script src="./matter.min.js"></script>
```

## 🎯 游戏玩法说明

### 基本操作
1. **拖拽** - 鼠标拖拽小鸟向后拉
2. **瞄准** - 查看白色轨迹预测线
3. **发射** - 松开鼠标发射小鸟
4. **激活能力** - 点击飞行中的小鸟触发特殊能力

### 小鸟能力
- 🐦 **红色**: 普通攻击，无特殊能力
- 🐦 **黄色**: 点击加速冲刺（1.5倍速度）
- 🐦 **蓝色**: 点击分裂成三只小鸟
- 🐦 **白色**: 点击下蛋炸弹
- 🐦 **黑色**: 点击爆炸（大范围伤害）

### 获胜条件
- 消灭关卡中所有小猪
- 使用更少的小鸟获得更高分数
- 达到评分阈值获得星级奖励
