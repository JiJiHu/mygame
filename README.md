# 🎮 小游戏合集

一个包含三个经典小游戏的多功能网页游戏合集，支持贪吃蛇、五子棋和数独游戏。

## 🎯 游戏列表

### 1. 贪吃蛇 (Snake)
- 经典街机游戏
- 使用方向键或 WASD 控制移动
- 吃食物变长，碰到墙壁或自己游戏结束
- 支持移动端触摸控制
- 分数记录和最高分保存

### 2. 五子棋 (Gomoku)
- 经典策略游戏
- 支持双人对战和人机对战模式
- 简单AI对手，适合初学者
- 美观的棋盘和棋子设计

### 3. 数独 (Sudoku)
- 经典逻辑推理游戏
- 三个难度级别：简单、中等、困难
- 错误提示功能
- 提示功能帮助解题
- 键盘和数字键盘双控制
- 实时计时和错误计数

## 🚀 部署

### Vercel 部署

1. **准备 GitHub 仓库**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: 小游戏合集"
   ```

2. **推送到 GitHub**
   - 在 GitHub 创建新仓库
   - 添加远程仓库并推送

3. **在 Vercel 部署**
   - 访问 [vercel.com](https://vercel.com)
   - 点击 "Add New Project"
   - 导入你的 GitHub 仓库
   - 点击 "Deploy"

### 本地运行

直接在浏览器中打开 `index.html` 文件即可运行。
